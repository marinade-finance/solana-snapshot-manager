# Marinade Snapshot API

## Installation

```bash
$ pnpm install
```

## Running the CLI
```bash
pnpm run cli parse --sqlite <input-sqlite> --output <output-csv>
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
